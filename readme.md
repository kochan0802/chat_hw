# 一人旅ライブ中継
## DEMO
  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - カメラを起動して、旅を中継する

  - それをみながら視聴者がコメントする（ニコ生風に流す）

## 工夫した点

  - カメラを起動させる
  
## 苦戦した点

  - ニコ生風コメントが自動再生と好きな文字を入れられなかった（土曜日までに改善する！）
  - ライブ配信したかったがやり方がわからなかった
